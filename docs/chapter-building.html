Chapter building

Using the utility files in the utils.R we point queryChapter to the relevant package and filter a pattern along a given field (attribute) of the help file (name, concept, title). This gives a data.frame of functions that meet the search criteria. We can then use this data.frame to create chapters for the gitbook using createChapter.

In this example a file named 01-Figures.Rmd will be created with a uniform output, a section with the function as its name and three chunks:
